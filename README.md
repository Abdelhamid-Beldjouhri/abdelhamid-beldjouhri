<h2>🔐 Cybersecurity Project</h2>

<h3>🛡 Cybersecurity Risk Assessment Report</h3>

<p><strong>Company:</strong> Botium Toys<br>
<strong>Prepared by:</strong> Abdelhamid Beldjouhri<br>
<strong>Date:</strong> 21/04/2025</p>

---

<h3>📌 Overview</h3>
<p>
This project presents a cybersecurity risk assessment of Botium Toys. The objective is to identify security gaps, evaluate risks, and recommend improvements based on industry best practices, including the NIST Cybersecurity Framework (CSF).
</p>

---

<h3>⚠️ Risk Level</h3>
<p><strong>High (8/10)</strong></p>

---

<h3>✅ Strengths</h3>
<ul>
<li>Firewall implemented to control network traffic</li>
<li>Antivirus software actively monitored</li>
<li>Physical security measures (locks, CCTV, fire systems)</li>
<li>Basic privacy policies established</li>
</ul>

---

<h3>❌ Weaknesses</h3>
<ul>
<li>No data backup or recovery plan</li>
<li>No encryption for sensitive data</li>
<li>Weak access control (excessive user permissions)</li>
<li>No Intrusion Detection System (IDS)</li>
<li>Weak password policies</li>
<li>Lack of asset management and classification</li>
</ul>

---

<h3>🚀 Recommendations</h3>
<ul>
<li>Implement encryption for sensitive data (at rest and in transit)</li>
<li>Establish regular backups and disaster recovery plan</li>
<li>Apply least privilege principle</li>
<li>Implement RBAC and MFA</li>
<li>Deploy IDS</li>
<li>Classify organizational assets</li>
<li>Regular updates and patching</li>
<li>Align with NIST CSF</li>
</ul>

---

<h3>🧾 Conclusion</h3>
<p>
Botium Toys demonstrates basic security measures; however, critical gaps significantly increase its exposure to cyber threats. Addressing these weaknesses is essential to improve the organization's security posture.
</p>
