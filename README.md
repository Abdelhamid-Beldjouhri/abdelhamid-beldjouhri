### 🛡 Cybersecurity Risk Assessment Report

**Company:** Botium Toys
**Prepared by:** Abdelhamid Beldjouhri
**Date:** 21/04/2025

---

### 📌 Overview

This project presents a cybersecurity risk assessment of Botium Toys. The objective is to identify security gaps, evaluate risks, and recommend improvements based on industry best practices, including the NIST Cybersecurity Framework (CSF).

---

### ⚠️ Risk Level

**Overall Risk Rating:** High (8/10)

---

### ✅ Strengths

* Firewall implemented
* Antivirus monitoring
* Physical security (CCTV, locks)
* Basic privacy policies

---

### ❌ Weaknesses

* No backup plan
* No encryption
* Weak access control
* No IDS
* Weak passwords
* No asset classification

---

### 🚀 Recommendations

* Implement encryption
* Establish backups & recovery plan
* Apply RBAC + MFA
* Deploy IDS
* Manage assets properly
* Follow NIST CSF

---

### 🧾 Conclusion

The company has basic security controls but critical gaps increase cyber risk. Improvements are necessary to strengthen security posture.
